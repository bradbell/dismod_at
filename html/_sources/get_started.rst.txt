.. _get_started@Run One Example:

Run One Example
***************
If you have
:ref:`built dismod_at<install_unix@dismod_at@Build>` and
are in the directory
:ref:`dismod_at.git<install_unix@Download dismod_at.git>` ,
you can run one of these examples with the command

   *python3_executable* ``example/get_started/`` *name*

where :ref:`run_cmake.sh@python3_executable`
is the python3 program and *name* is the name of the example.
The name of the example
includes the ``.py`` extension at the end.

.. _get_started@get_started.db:

get_started.db
**************
Each of these examples uses the database

   ``build/example/get_started/get_started.db``

which is created by :ref:`get_started_db.py-name` and
can be inspected using any sqlite viewer; e.g.,
http://sqlitebrowser.org/.
