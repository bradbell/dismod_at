C++: Get the Density Table Information

Syntax
Purpose
db
density_table
density_enum
log_density
density_enum2name
Example

Syntax
density_table = get_density_table(db)
 is_log = log_density(density)

Purpose
To read the density_table and return it as a C++ data structure.

db
The argument db has prototype
     sqlite3* db
 and is an open connection to the database.

density_table
The return value density_table has prototype
     CppAD::vector<density_enum>  density_table
 For each density_id ,
     density_table[density_id]
 is the enum value for the corresponding density_name .

density_enum
This enum type has the following values:
value    density_name
uniform_enum    uniform
gaussian_enum    gaussian
c_gaussian_enum    c_gaussian
log_gaussian_enum    log_gaussian
laplace_enum    laplace
log_laplace_enum    log_laplace
students_enum    students
log_students_enum    log_students
The number of these enum values is number_density_enum.

log_density
The return value is_log is true if the corresponding density is log_gaussian, log_laplace, or log_students. It is false otherwise.

density_enum2name
This is a global variable. If density , is an density_enum value, density_enum2name[density] is the density_name corresponding to the enum value.

Example
The file get_density_table_xam.cpp contains an example that uses this function.
