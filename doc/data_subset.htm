See Also
avgint_subset .

Purpose
This routine subsamples the data_table , in the following way:
  1. Only rows corresponding to the parent_node , or a descendant of the parent node, are included.
  2. Only rows for which the covariates satisfy the max_difference criteria are included.
  3. The subsampled rows are the same as the corresponding original row except that for each covariate, its reference value is subtracted from the value of the covariate in data_table .
  4. All of the child data is checked to make sure that it does not use a laplace or log_laplace density.


data_table
This argument has prototype
     const CppAD::vector<data_struct>& data_table
 and is the data_table .

data_cov_value
This argument has prototype
     const CppAD::vector<double>& data_cov_value
 and is the data_table covariate values.

covariate_table
This argument has prototype
     const CppAD::vector<covariate_struct>& covariate_table
 and is the covariate_table .

child_object
This argument has prototype
     const child_info& child_object
data_subset_obj
This argument has prototype
     CppAD::vector<data_subset_struct>& data_subset_obj
 Its input size is zero and upon return its size is the number of rows in data_table that satisfy the purpose above. The structure has all the fields that are present in data_struct .

n_subset
We use the notation n_subset = data_subset_obj.size() .

subset_id
We use the notation subset_id for an index between zero and n_subset-1 ,

original_id
There an extra field in data_struct that has name original_id, type int. The values in this field are equal to the original_id for the corresponding row of data_table . The value of
     data_subset_obj[subset_id].original_id
 increases with subset_id ; i.e., for each subset_id less than n_subset-2 ,
     data_subset_obj[subset_id].original_id <
          data_subset_obj[subset_id+1].original_id
data_subset_cov_value
This argument has prototype
     CppAD::vector<double>& data_subset_cov_value
 Its input size is zero and upon return its size is n_subset * n_covariate . For each subset_id and covariate_id ,
data_subset_cov_value[subset_id * n_covariate + covariate_id]
= data_cov_value[original_id * n_covariate + covariate_id]
  - reference(covariate_id)
 where
     original_id = data_subset_obj[subset_id].original_id
 and reference(covariate_id) is the reference value for the corresponding covariate_id . Note that if the max_difference value is null in the covariate table, or the covariate value is null in data_table ,
data_subset_cov_value[subset_id * n_covariate + covariate_id] = 0
 Also note that
| data_subset_cov_value[subset_id * n_covariate + covariate_id] |
<= max_difference(covariate_id)
 where max_difference(covariate_id) is the maximum difference for the corresponding covariate_id .
