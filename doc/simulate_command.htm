The Simulate Command

Syntax
dismod_at database simulate number_simulate

database
Is an http://www.sqlite.org/sqlite/ database containing the dismod_at input tables which are not modified.

number_simulate
Is the number of simulations. Each simulation contains a complete data set.

truth_var_table
The truth_var_table is an addition input table for this command. It specifies the true values for the model_variables used during the simulation. This table can be create by the set_command , or the user can create it directly with the aid of the var_table (created by the init_command ).

data_table
It the data density is log_gaussian or log_laplace, meas_value is used to log-transform the standard deviations. If the density is gaussian or laplace, the simulation standard deviation is the same as meas_std and negative simulated measurements are not converted to zero.

simulate_table
A new simulate_table is created. It contains simulated measurement values meas_value and standard deviations meas_std . These values can be used in place of the data table values for meas_value and meas_std . Only the data_id that are in the data_subset table are included in the simulated measurements. Hence the number of rows in simulate_table is number_simulate times the number of rows in data_subset_table .

meas_value
Negative simulated data values are converted to zero. Hence meas_value in the simulate table are all non-negative.

meas_std
It the data density is uniform, gaussian, or laplace, meas_std in the simulate table is set so that the adjusted standard deviation delta is the same as for the data table. Note that a small minimum is chosen if the condition above would require meas_std to be near zero or negative.

It the data density is log_gaussian or log_laplace, meas_std in the simulate table is set so that the log-transformed standard deviation sigma is the same as for the data table. Note that the adjusted standard deviation is used in the log transformation. Also note that a small minimum is chosen if the condition above would require meas_std to be near zero or negative.
