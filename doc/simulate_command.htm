The Simulate Command

Syntax
dismod_at database simulate number_simulate

database
Is an http://www.sqlite.org/sqlite/ database containing the dismod_at input tables which are not modified.

number_simulate
Is the number of simulations. Each simulation contains a complete data set.

truth_var_table
The truth_var_table is an addition input table for this command. It specifies the values for the model_variables used during the simulation. This table can be create by the set_command , or the user can create it directly with the aid of the var_table (created by the init_command ).

simulate_table
A new simulate_table is created by this command. It contains simulated measurements and adjusted standard deviations. Only the data_id that are in the data_subset table are included in the simulated measurements. In addition, number_simulate values are simulated for such data_id . Hence the number of rows in simulate_table is number_simulate times the number of rows in data_subset_table .

simulate_value
The column contains the value of y for each subset_data_id and each simulate_index .

simulate_delta
The column contains the value of y for each subset_data_id and each simulate_index . Note that the value of @(@ \delta @)@ only depends on data_subset_id .

d
We use @(@ d @)@ to denote the density_id and eta corresponding to this data_id .

delta
We use @(@ \delta @)@ to denote the adjusted standard deviation corresponding to this data_id .

A
We use @(@ A @)@ denote the average integrand corresponding to the value for the model variables and this data_id .

y

Linear
If @(@ d @)@ is Gaussian, Laplace, or Students, @[@ y = A + e @]@ where @(@ e @)@ is simulated from the corresponding distribution with mean zero and standard deviation @(@ \delta @)@.

Log-Transformed
If @(@ d @)@ is Log-Gaussian, Log-Laplace, or Log-Students, @[@ e = \log( y + \eta ) - \log( A + \eta ) @]@ @(@ e @)@ is simulated from the corresponding linear distribution with mean zero and standard deviation @(@ \delta @)@. It follows that @[@ y = \max\{ 0 , \exp(e) * ( A + \eta ) - \eta \} @]@

Example
The file simulate_command.py contains an example and test using this command.
