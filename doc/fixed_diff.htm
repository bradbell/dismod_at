k, K
For each rate_id there is parent_smooth_id . In addition, for each mulcov_id there is a smooth_id . We use @(@ k @)@ to index all the rates and covariates multipliers and @(@ K @)@ for the number of rates plus the number of covariate multipliers.

I(k)
We use @(@ I(k) @)@ to denote the number of age points in the smoothing with index @(@ k @)@.

J(k)
We use @(@ J(k) @)@ to denote the number of time points in the smoothing with index @(@ k @)@.

lambda_k
We use @(@ \lambda_k^a @)@ ( @(@ \lambda_k^t @)@ ) to denote the mulstd_dage_prior_id ( mulstd_dtime_prior_id ) for smooth_id corresponding to @(@ k @)@.

theta_i,j,k
Let @(@ \theta_{i,j,k} @)@ denote the fixed effect corresponding to rate or covariate index @(@ k @)@, age index @(@ i @)@ and time index @(@ j @)@. Note that if @(@ k @)@ corresponds to a rate, it is the corresponding parent rates that are fixed effects.

mu_i,j,k
We use @(@ \mu_{i,j,k}^a @)@ ( @(@ \mu_{i,j,k}^t @)@ ) to denote the mean corresponding to the dage_prior_id ( dtime_prior_id ) rate or covariate index @(@ k @)@, age index @(@ i @)@ and time index @(@ j @)@.

delta_i,j,k
We use @(@ \delta_{i,j,k}^a @)@ ( @(@ \delta_{i,j,k}^t @)@ ) to denote the std corresponding to the dage_prior_id ( dtime_prior_id ) rate or covariate index @(@ k @)@, age index @(@ i @)@ and time index @(@ j @)@.

d_i,j,k
We use @(@ d_{i,j,k}^a @)@ ( @(@ d_{i,j,k}^t ) @)@ to denote the density_id and eta corresponding to dage_prior_id ( dtime_prior_id ) rate or covariate index @(@ k @)@, age index @(@ i @)@ and time index @(@ j @)@.

Age Differences
For @(@ k = 0 , \ldots , K - 1 @)@, @(@ i = 0, \ldots , I(k)-2 @)@, and @(@ j = 0, \ldots , J(k)-1 @)@, the age difference density for grid index @(@ (i, j) @)@ and smoothing index @(@ k @)@ is @[@ A_{i,j,k}^\theta ( \theta ) = \exp \left[ D \left( \theta_{i+1,j,k} \W{,} \theta_{i,j,k} \W{,} \mu_{i,j,k}^a \W{,} \lambda_k^a \delta_{i,j,k}^a \W{,} d_{i,j,k}^a \right) \right] @]@ where @(@ D @)@ is the difference log-density function .

A^theta
The age difference density for the entire fixed effects vector @(@ \theta @)@ is defined by @[@ A^\theta ( \theta ) = \prod_{k=0}^{K-1} \prod_{i=0}^{I(k)-2} \prod_{j=0}^{J(k)-1} A_{i,j,k}^\theta ( \theta ) @]@

Time Differences
For @(@ k = 0 , \ldots , K - 1 @)@, @(@ i = 0, \ldots , I(k)-1 @)@, and @(@ j = 0, \ldots , J(k)-2 @)@, the time difference density for grid index @(@ (i, j) @)@ and smoothing index @(@ k @)@ is @[@ T_{i,j,k}^\theta ( \theta ) = \exp \left[ D \left( \theta_{i,j+1,k} \W{,} \theta_{i,j,k} \W{,} \mu_{i,j,k}^t \W{,} \lambda_k^t \delta_{i,j,k}^t \W{,} d_{i,j,k}^t \right) \right] @]@

T^theta
The time difference density for the entire fixed effects vector @(@ \theta @)@ is defined by @[@ T^\theta ( \theta ) = \prod_{k=0}^{K-1} \prod_{i=0}^{I(k)-1} \prod_{j=0}^{J(k)-2} T_{i,j,k}^\theta ( \theta ) @]@
