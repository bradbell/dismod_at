fit_object
The fit_model object being constructed.

db
This argument is the database connection for logging errors and warnings.

bound_random
This is the value of the bound_random in the option table.

fit_or_sample
This argument is either fit or sample. If it is fit (sample) the run_fit ( sample_posterior ) function can be used with this object.

pack_object
This argument is the pack_info information corresponding to the model_variables .

start_var
This vector is the starting model_variables in the order specified by pack_info .

prior_table
This argument is the prior_table .

s_info_vec
For each smooth_id ,
     s_info_vec[ smooth_id ]
 is the corresponding smooth_info information.

data_object
This object contains the model for the data density; see data_model .

prior_object
This object contains the model for the fixed negative log-likelihood; see prior_model .

quasi_fixed
If this argument is true, a quasi-Newton method is used when optimizing the fixed effects. Otherwise a full Newton method is used; see quasi_fixed .

zero_sum_random
If this vector has size number_rate_enum. If zero_sum_random[rate_id] is true, for each age, time, the sum of the random effects for the corresponding rate is constrained to be zero.
