C++: Get the nslist Table Information

Syntax
nslist_table = get_nslist_table(db)

Purpose
To read the nslist_table and return it as a C++ data structure.

db
The argument db has prototype
     sqlite3* db
 and is an open connection to the database.

nslist_table
The return value nslist_table has prototype
     CppAD::vector<std::string>  nslist_table
 For each nslist_id ,
     nslist_table[nslist_id]
 is the name for the corresponding nslist_id .

Example
The file get_nslist_table_xam.cpp contains an example that uses this function.
