Some Statistical Function Definitions

Notation
@(@ y @)@ quantity that we are computing a statistical density for
@(@ z @)@ quantity we are differencing from @(@ y @)@
@(@ \mu @)@ expected value @(@ y @)@, difference, or difference of logs
@(@ \delta @)@ standard deviation for @(@ y @)@, difference, or difference of logs
@(@ d:id @)@ density_id
@(@ d:\eta @)@ eta the offset in log transform
@(@ d:\nu @)@ nu degrees of freedom in Student's-t.

Log-Transformed Standard Deviation, sigma
We define the log-transformed standard deviation by @[@ \sigma( \mu, \delta , \eta ) = \log ( \mu + \eta + \delta ) - \log ( \mu + \eta ) @]@

Weighted Residual Function, R

Value
The weighted residual function for values @[@ R(y, \mu, \delta, d) @]@ is defined separately below for each possible value of the density name.

Difference
The weighted residual function for differences @[@ R(z, y, \mu, \delta, d) @]@ has extra argument @(@ z @)@ and is also is defined separately below for each possible value of the density name.

Uniform
If the density name is uniform, the weighted residual functions are @[@ R(y, \mu, \delta, d) = 0 @]@ @[@ R(z, y, \mu, \delta, d) = 0 @]@

Gaussian, Laplace, Student's-t
If the density name is gaussian, laplace, or students. the weighted residual functions are @[@ R(y, \mu, \delta, d) = \frac{y - \mu}{\delta} @]@ @[@ R(z, y, \mu, \delta, d) = \frac{z - y - \mu}{\delta} @]@

Log-Gaussian, Log-Laplace, Log-Student's-t
If the density name is log_gaussian, log_laplace, or log_students, the weighted residual functions are @[@ R(y, \mu, \delta, d) = \frac{ \log( y + d:\eta ) - \log( \mu + d:\eta ) }{ \sigma( \mu , \delta , d:\eta ) } @]@ @[@ R(z, y, \mu, \delta, d) = \frac{ \log(z + d:\eta ) - \log( y + d:\eta ) - \mu }{ \delta } @]@

Log-Density Function, D
The log-density function @[@ D(y, \mu, \delta, d) @]@ is defined separately below for each possible value of the density name.

Difference
The log-density function for differences @[@ D(z, y, \mu, \delta, d) @]@ has extra argument @(@ z @)@ and is also is defined separately below for each possible value of the density name.

Uniform
If the density name is uniform, the log-density functions @(@ D(y, \mu, \delta, d) @)@, and @(@ D(z, y, \mu, \delta, d) @)@ are define by @(@ D = 0 @)@.

Gaussian
If the density name is gaussian, the log-density functions @(@ D(y, \mu, \delta, d) @)@, and @(@ D(z, y, \mu, \delta, d) @)@ are define by @[@ D = - \log \left( \delta \sqrt{2 \pi} \right) - \frac{1}{2} R^2 @]@ where @(@ D @)@ and @(@ R @)@ have the same arguments; see Weighted Residual Function, R .

Laplace
If the density name is laplace, the log-density functions @(@ D(y, \mu, \delta, d) @)@, and @(@ D(z, y, \mu, \delta, d) @)@ are define by @[@ D = - \log \left( \delta \sqrt{2} \right) - \sqrt{2} | R | @]@ where @(@ D @)@ and @(@ R @)@ have the same arguments.

Student's-t
If the density name is students, the log-density functions @(@ D(y, \mu, \delta, d) @)@, and @(@ D(z, y, \mu, \delta, d) @)@ are define by @[@ D = \log \left( \frac{ \Gamma( ( \nu + 1 ) / 2 ) }{ \sqrt{ \nu \pi } \Gamma( \nu / 2 ) } \right) - \frac{\nu + 1}{2} \log \left( 1 + R^2 / ( \nu - 2 ) \right) @]@ where @(@ D @)@ and @(@ R @)@ have the same arguments.

Log-Gaussian
If the density name is log_gaussian, the log-density functions are defined by @[@ D(y, \mu, \delta, d) = - \log \left[ \sigma( \mu , \delta , d:\eta ) \sqrt{2 \pi} \right] - \frac{1}{2} R(y, \mu, \delta, d)^2 @]@ @[@ D(z, y, \mu, \delta, d) = - \log \left( \delta \sqrt{2 \pi} \right) - \frac{1}{2} R(z, y, \mu, \delta, d)^2 @]@ see Log-Transformed standard deviation .

Log-Laplace
If the density name is log_laplace, the log-density functions are @[@ D(y, \mu, \delta, d) = - \log \left[ \sigma( \mu , \delta , d:\eta ) \sqrt{2} \right] - \sqrt{2} \left| R(y, \mu, \delta, d) \right| @]@ @[@ D(z, y, \mu, \delta, d) = - \log \left( \delta \sqrt{2} \right) - \sqrt{2} \left| R(z, y, \mu, \delta, d) \right| @]@

Log-Student's-t
If the density name is log_students, the log-density functions @(@ D(y, \mu, \delta, d) @)@, and @(@ D(z, y, \mu, \delta, d) @)@ are define by @[@ D = \log \left( \frac{ \Gamma( ( \nu + 1 ) / 2 ) }{ \sqrt{ \nu \pi } \Gamma( \nu / 2 ) } \right) - \frac{\nu + 1}{2} \log \left( 1 + R^2 / ( \nu - 2 ) \right) @]@ where @(@ D @)@ and @(@ R @)@ have the same arguments.
