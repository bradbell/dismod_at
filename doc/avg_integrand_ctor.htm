Prototype 

avg_integrand::avg_integrand(
          double                                    ode_step_size    ,
          const std::string&                        rate_case        ,
          const CppAD::vector<double>&              age_avg_grid     ,
          const CppAD::vector<double>&              age_table        ,
          const CppAD::vector<double>&              time_table       ,
          const CppAD::vector<integrand_struct>&    integrand_table  ,
          const CppAD::vector<weight_info>&         w_info_vec       ,
          const CppAD::vector<smooth_info>&         s_info_vec       ,
          const pack_info&                          pack_object      )

ode_step_size
This is the value of ode_step_size in the option table.

rate_case
This is the value of rate_case in the option table.

age_avg_grid
This is the age_avg_grid . A reference to age_avg_grid is used by avgint_obj ( age_avg_grid must no be deleted for as long as avgint_obj is used).

age_table
This argument is the age_table . A reference to age_table is used by avgint_obj ( age_table must no be deleted for as long as avgint_obj is used).

time_table
This argument is the time_table . A reference to time_table is used by avgint_obj .

integrand_table
This argument is the integrand_table . A reference to integrand_table is used by avgint_obj .

w_info_vec
For each weight_id ,
     w_info_vec[ weight_id ]
 is the corresponding weight_info information.

s_info_vec
For each smooth_id ,
     s_info_vec[ smooth_id ]
 is the corresponding smooth_info information. None of the prior information in s_info_vec is used. A reference to s_info_vec is used by avgint_obj .

pack_object
This is the pack_info information corresponding to the model_variables .
