C++: Get the List of Smoothing Table

Syntax
Purpose
db
nslist_pair
nslist_pair_struct
Example

Syntax
nslist_pair = get_nslist_pair_table(db)

Purpose
To read the nslist_pair_table and return it as a C++ data structure.

db
The argument db has prototype
     sqlite3* db
 and is an open connection to the database.

nslist_pair
The return value nslist_pair has prototype
     CppAD::vector<nslist_pair_struct> nslist_pair
 For each nslist_pair_id ,
     nslist_pair_table[nslist_pair_id]
 is the information for one entry in one smoothing list.

nslist_pair_struct
This is a structure with the following fields
int density_id nslist_id identifies this list of smoothings.
int node_id node_id is the node that this smoothing (in this list) applies to.
int smooth_id smooth_id is the smoothing that is applied to this node (for this list).

Example
The file get_nslist_pair_xam.cpp contains an example that uses this function.
