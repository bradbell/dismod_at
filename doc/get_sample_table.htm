C++: Get the Simulate Table

Syntax
Purpose
db
sample_table
sample_struct
Example

Syntax
sample_table = get_sample_table(db)

Purpose
To read the sample_table and return it as a C++ data structure.

db
The argument db has prototype
     sqlite3* db
 and is an open connection to the database.

sample_table
The return value sample_table has prototype
     CppAD::vector<sample_struct>  sample_table
 For each sample_id ,
     sample_table[sample_id]
 is the information for the corresponding sample_id .

sample_struct
This is a structure with the following fields
Type Field Description int sample_index The sample_index for this sampled measurement.
int var_id The var_id for this sample
double var_value The var_value for this sample.

Example
The file get_sample_table_xam.cpp contains an example and test of this function.
