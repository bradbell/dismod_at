Set Value Necessary for Likelihood (not for Average Integrand)

Syntax
data_object.replace_like(minimum_meas_cv, data_subset_obj)

Purpose
The values density_id , hold_out , meas_value , meas_std are not necessary to calculate data_object.avg_no_ode and data_object.avg_yes_ode . However, the are necessary to use the functions data_object.like_one and data_object.like_all .

data_object
This object has prototype
     data_model data_object
 see data_object constructor .

minimum_meas_cv
This argument has prototype
     double minimum_meas_cv
 and is the minimum_meas_cv . The minimum_meas_cv_ value in data_object is replaced by minimum_meas_cv .

subset_object
We use subset_object for the subset_object in the data_object constructor.

n_subset
We use the notation
     n_subset = subset_object.size()
subset_id
This an index between zero and n_subset - 1 . It is used to refer to the corresponding element of subset_object .

data_subset_obj
This argument has prototype
     const CppAD::vector<data_subset_struct>& data_subset_obj
 and has size n_subset . For each subset_id , and for field = density_id, hold_out, meas_value, meas_std , data_subset_obj[subset_id].field , is used as a replacement for subset_object[subset_id].field .
