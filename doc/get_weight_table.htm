C++: Get the Weight Table Information

Syntax
weight_table = get_weight_table(db)

Purpose
To read the weight_table and return it as a C++ data structure.

db
The argument db has prototype
     sqlite3* db
 and is an open connection to the database.

weight_table
The return value weight_table has prototype
     CppAD::vector<weight_struct>  weight_table
 For each weight_id ,
     weight_table[weight_id]
 is the information for the corresponding weight_id .

weight_struct
This is a structure with the following fields
Type Field Description
std::string weight_name The weight_name for this weighting.
std::string n_age The n_age for this weighting.
std::string n_time The n_time for this weighting.

Example
The file get_weight_grid_xam.cpp contains an example that uses this function.
