template <class Float>
CppAD::vector<Float> adj_integrand::line(
     const CppAD::vector<double>&                       line_age         ,
     const CppAD::vector<double>&                       line_time        ,
     size_t                                             integrand_id     ,
     size_t                                             n_child          ,
     size_t                                             child            ,
     const CppAD::vector<double>&                       x                ,
     const CppAD::vector<Float>&                        pack_vec         ,
rate_case
This is the value of rate_case in the option table.

age_table
This argument is the age_table . A reference to age_table is used by adjint_obj ( age_table must no be deleted for as long as adjint_obj is used).

time_table
This argument is the time_table . A reference to time_table is used by adjint_obj .

integrand_table
This argument is the integrand_table . A reference to integrand_table is used by adjint_obj .

s_info_vec
For each smooth_id ,
     s_info_vec[ smooth_id ]
 is the corresponding smooth_info information. None of the prior information in s_info_vec is used. A reference to s_info_vec is used by adjint_obj .

pack_object
This is the pack_info information corresponding to the model_variables .

line_age
This vector is the age points at which the adjusted integrand is computed. It is faster if successive points have close values in age; e.g.,
     line_age[k] <= line_age[k+1]
n_line
We use the notation n_line = line_age.size() for the number points at which the approximate solution is returned.

line_time
This vector has size n_line and is the time points at which the adjusted integrand is computed. It is faster if successive points have close values in time; e.g.,
     line_time[k] >= line_time[k+1]
ODE
In the case where the integrand (specified by integrand_id ) requires solving the ODE , the line must be a cohort; i.e., it must satisfy the following properties:
  1. The first line age must be the minimum value in the age table; i.e.,
             line_age[0] = min_i table_age[i]
  2. The line ages must be monotone increasing; i.e., for k = 1 , ..., n_line-1
             line_age[k-1] < line_age[k]
  3. The difference between the line ages and times is constant; i.e., for k = 1 , ..., n_line-1
             line_time[k] - line_age[k] == line_time[0] - line_age[0]
In this case cohort_age and cohort_time are better names for the arguments line_age and line_time .

integrand_id
This is the integrand_id in the integrand table.

n_child_
set to number of children .

child
Is the child corresponding to this adjustment of the integrand.

pack_vec
is all the model_variables in the order specified by pack_object .

Float
The type Float must be double or a1_double .

adj_line
The return value is a vector with size n_line and adj_line[i] is the adjusted integrand at age line_age[i] and time line_time[i] .
