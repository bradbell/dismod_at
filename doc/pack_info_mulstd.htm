Devel Pack Variables: Standard Deviations Multipliers

Syntax
pack_object
smooth_id

     offset
Example

Syntax
offset = pack_object.mulstd_offset(smooth_id, k)
pack_object
This object has prototype
     const pack_info pack_object
smooth_id
This argument has prototype
     size_t smooth_id
 and is the smooth_id for this multiplier.

k
This argument has prototype
     size_t k
 It specifies which type of priors standard deviations that get multiplied by this variable as follows:
k    Type of prior
0    value priors for this smoothing
1    age difference priors for this smoothing
2    time difference priors for this smoothing

offset
The return value has prototype
     size_t offset
 and is the offset (index) in the packed variable vector where this multiplier is located. If it has value
     DISMOD_AT_NULL_SIZE_T
 This multiplier has value one and is not a variable.
