Under Construction
This is a plan for censored distribution functions, but they have not yet been implemented.

Discussion
We use @(@ \mu \geq 0 @)@ for the mean and @(@ \delta > 0 @)@ for the standard deviation of a Gaussian or Laplace data @(@ y @)@. The crucial property is that the censored probability functions defined below are smooth function with respect to the mean value @(@ \mu @)@ (but not even continuous with respect to the data @(@ y @)@).

Gaussian

Density
The Gaussian density function is given by @[@ G( y ; \mu , \delta ) = \sqrt{ \frac{1}{ 2 \pi \delta^2 } } \exp \left[ - \frac{1}{2} \left( \frac{y - \mu}{\delta} \right)^2 \right] @]@

Error Function
The Error function is defined by @[@ \R{erf}(x) = \sqrt{ \frac{1}{\pi} } \int_{-x}^{+x} \exp \left( - t^2 \right) \; \R{d} t @]@ Using he change of variables @(@ t = (y - \mu) / ( \delta \sqrt{2} ) @)@ we have @[@ \R{erf}(x) = \sqrt{ \frac{1}{2 \pi \delta^2} } \int_{\mu - x}^{\mu + x} \exp \left[ - \frac{1}{2} \left( \frac{y - \mu}{\delta} \right)^2 \right] \; \R{d} y @]@ Setting @(@ x = \mu @)@ we obtain @[@ \R{erf}(\mu) = \sqrt{ \frac{1}{2 \pi \delta^2} } \int_{0}^{2 \mu} \exp \left[ - \frac{1}{2} \left( \frac{y - \mu}{\delta} \right)^2 \right] \; \R{d} y @]@ using the fact the Gaussian density is symmetric about the value @(@ y = \mu @)@, and its integral from minus infinity to plus infinity is one, we obtain @[@ \frac{1 - \R{erf}(\mu) }{2} = \sqrt{ \frac{1}{2 \pi \delta^2} } \int_{-\infty}^{0} \exp \left[ - \frac{1}{2} \left( \frac{y - \mu}{\delta} \right)^2 \right] \; \R{d} y @]@

Censored Probability, G_+
Suppose that @(@ y \sim N( \mu, \delta^2 ) @)@ and define @(@ y_+ = \R{max}(x, 0) @)@. We define the censored Gaussian probability by @[@ G_+ ( y_+ ; \mu , \delta ) = \left\{ \begin{array}{ll} [ 1 - \R{erf}(\mu) ] / 2 & \R{if} \; y_+ = 0 \\ G( y_+ ; \mu , \delta ) & \R{otherwise} \end{array} \right. @]@

Laplace

Density
The Laplace density function is given by @[@ L( y ; \mu , \delta ) = \delta \sqrt{ \frac{1}{2} } \exp \left[ - \sqrt{2} \left| \frac{y - \mu}{\delta} \right| \right] @]@

Indefinite Integral
The indefinite integral with respect to @(@ y @)@, for @(@ x \leq \mu @)@, is @[@ \int_{-\infty}^{x} L( y ; \mu , \delta ) \; \R{d} y = \frac{1}{2} \exp \left( - \sqrt{2} \frac{ \mu - x}{\delta} \right) @]@ Using @(@ 0 \leq \mu @)@, we obtain @[@ \int_{-\infty}^{0} L( y ; \mu , \delta ) \; \R{d} y = \frac{1}{2} \exp \left( - \sqrt{2} \frac{\mu}{\delta} \right) @]@

Censored Probability, L_+
Suppose that @(@ y \sim L( \mu, \delta^2 ) @)@ and define @(@ y_+ = \R{max}(x, 0) @)@. We define the censored Laplace probability by @[@ L_+ ( y_+ ; \mu , \delta ) = \left\{ \begin{array}{ll} (1 / 2 ) \exp \left( - \mu \sqrt{2} / \delta \right) & \R{if} \; y_+ = 0 \\ L( y_+ ; \mu , \delta ) & \R{otherwise} \end{array} \right. @]@
