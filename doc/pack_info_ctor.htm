 size            = pack_object.size()
 integrand_size  = pack_object.integrand_size()
 child_size      = pack_object.child_size()
n_integrand
This argument has prototype
     size_t n_integrand
 and is the number of integrands; i.e., the size of integrand_table . If mulcov_table has size zero, then n_integrand can be zero (a case used for testing purposes).

child_id2node_id
This argument has prototype
     const CppAD::vector<size_t> child2node
 and is a mapping from child_id to node_id ; see child_id2node_id . The size of this vector is the number of children; see child_size . If
     rate_table[rate_id].child_nslist_id == DISMOD_AT_NULL_INT
 for all rate_id , only the size of this vector matters (its values are not used).

smooth_table
This argument has prototype
     const CppAD::vector<smooth_struct>& smooth_table
 and is the smooth_table . Only the following fields of this table are used: n_age, n_time.

mulcov_table
This argument has prototype
     const CppAD::vector<mulcov_struct>& mulcov_table
 and is the mulcov_table .

rate_table
This argument has prototype
     const CppAD::vector<rate_struct>& rate_table
 and is the rate_table .

nslist_pair
This argument has prototype
     const CppAD::vector<nslist_pair_struct>& nslist_pair
 and is the nslist_pair . If
     rate_table[rate_id].child_nslist_id == DISMOD_AT_NULL_INT
 for all rate_id , this table is not used and can be empty; i.e., have size zero.

pack_copy
This object is a copy of the pack_object object (and acts the same).

size
This function is const. Its return value has prototype
     size_t size
 and is the total number of variables; i.e., the number of elements in the packed variable vector.

integrand_size
This function is const. Its return value has prototype
     size_t integrand_size
 and is the value of n_integrand in the constructor.

child_size
This function is const. Its return value has prototype
     size_t child_size
 and is the value of child_id2node_id.size() .

smooth_size
This function is const. Its return value has prototype
     size_t smooth_size
 and is the value is smooth_table.size() in the constructor.

random_size
This function is const. Its return value has prototype
     size_t random_size
 and is the number of random effects in the model (counting those that are constrained with equal upper and lower limits).

Age-Time Order
When an function of age and time is stored with a specified offset in a packed vector, it is in time major order; i.e., for i = 0, ... , n_age-1 , for j = 0, ... , n_time-1 ,
     offset + i * n_time + j
 is the index in the packed vector of the corresponding age-time point.
