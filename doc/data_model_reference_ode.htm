Solve for S and C Corresponding to Reference Covariate Values

Syntax
reference_sc = data_object.reference_sc(pack_vec, parent_only)

Prototype 

template <class Float>
CppAD::vector<Float> data_model::reference_ode(
     const CppAD::vector<Float>& pack_vec     ,
     bool                        parent_only
) const

data_object
see data_object constructor .

Float
The type Float must be one of the following: double or one of the ad_types .

pack_vec
is all the model_variables in the order specified by pack_info .

parent_only
if true, only the parent values in reference_sc are calculated ( c = n_child_ ). The other elements in reference_sc are not defined.

reference_sc
The return value has prototype
     const CppAD::vector<Float>& reference_sc
 It's size is given by
     reference_sc.size() = 2 * (n_child_ + 1) * n_ode
 Let n_ode = n_age_ode_ * n_time_ode_ . For c = 0, ..., n_child_ , for i = 0, ..., n_age_ode_-1 , for j = 0, ..., n_time_ode_-1 ,
     reference_sc[ 2*(child * n_ode + i * n_time_ode_ + j) + 0]
 is the susceptible fraction S and
     reference_sc[ 2*(child * n_ode + i * n_time_ode_ + j) + 1]
 is the with condition fraction C corresponding to the specified child c and reference value for the rate covariates. The child c = n_child_ corresponds to the parent. If c < n_child_ and %child_ran_zero_[%c%]% is true, the corresponding values are not calculated because c = n_child_ can be used instead.
