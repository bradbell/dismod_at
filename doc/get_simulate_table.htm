C++: Get the Simulate Table

Syntax
simulate_table = get_simulate_table(db)

Purpose
To read the simulate_table and return it as a C++ data structure.

db
The argument db has prototype
     sqlite3* db
 and is an open connection to the database.

simulate_table
The return value simulate_table has prototype
     CppAD::vector<simulate_struct>  simulate_table
 For each simulate_id ,
     simulate_table[simulate_id]
 is the information for the corresponding simulate_id .

simulate_struct
This is a structure with the following fields
Type Field Description
int simulate_index The simulate_index for this simulated measurement.
int data_subset_id The data_subset_id for this simulated measurement.
double meas_value The meas_value for this simulated measurement.
double meas_std The meas_std for this simulated measurement.

Example
The file get_simulate_table_xam.cpp contains an example and test of this function.
