C++: Get the Option Table Information

Syntax
option_table = get_option_table(db)

Purpose
To read the option_table and return it as a C++ data structure.

db
The argument db has prototype
     sqlite3* db
 and is an open connection to the database.

option_struct
This is a structure with the following fields
Type Field Description
std::string option_name The name for this option
std::string option_value The value for this option

option_table
The return value option_table has prototype
     CppAD::vector<option_struct>  option_table
 For i = 0 , ..., n_option ,
     option_table[i]
 is the information for the one option.

Special Note
This table is special because the index i is not the same as option_id in the option_table . This is because all the options have default values that are used when an option does not appear in the actual table.

Example
The file get_option_table_xam.cpp contains an example that uses this function.
