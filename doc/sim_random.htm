Syntax
z = sim_random(difference, density, mu, delta, eta, nu)

manage_gsl_rng
The routine manage_gsl_rng sets up and controls the underlying simulated random number generator.

difference
This argument has prototype
     bool difference
 If it is true, this simulation is for a difference, otherwise it is for a value; see delta below.

density
This argument has prototype
     density_enum density
 It specifies the density for the distribution that we are simulating. The following table specifies the meaning of this choice:
uniform_enum   not allowed
gaussian_enum    Gaussian
laplace_enum    Laplace
students_enum    Student's-t
log_gaussian_enum    Log-Gaussian
log_laplace_enum    Log-Laplace
log_students_enum    Log-Student's-t

mu
This argument has prototype
     double mu
 In the case were density is gaussian_enum, laplace_enum or students_enum, it is the mean for the distribution that we are simulating. Otherwise log( mu + eta ) is the mean of the log-transformed data.

delta
This argument has prototype
     double delta
 It is assumed delta is greater than zero and not infinity.

Linear
In the case were density is gaussian_enum, laplace_enum, students_enum, delta it is the standard deviation for z - mu .

difference
If the density is a log density, and difference is true, delta is the standard deviation for
     log( z + eta ) - log( mu + eta )
value
If the density is a log density, and difference is false,
     log( mu + eta + delta ) - log( mu + eta )
 is the standard deviation for
     log( z + eta ) - log( mu + eta )
eta
This argument has prototype
     double eta
 In the case were density is log_gaussian_enum, log_laplace_enum or log_students_enum, this is the offset in the log transformation for the distribution. In this case, it is assumed that mu + eta > 0 . Otherwise, eta is not used.

nu
This argument has prototype
     double nu
 In the case were density is students_enum or log_students_enum, it is the degrees of freedom in the Student's-t distribution. Otherwise it is not used.

z
The return value has prototype
     double z
 It simulates a sample from the specified distribution that is independent for the any previous return values.
