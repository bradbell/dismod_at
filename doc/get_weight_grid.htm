C++: Get the Weight Grid Information

Syntax
Purpose
db
weight_grid_struct
weight_grid
Example

Syntax
weight_grid = get_weight_grid(db)

Purpose
To read the weight_grid_table and return it as a C++ data structure.

db
The argument db has prototype
     sqlite3* db
 and is an open connection to the database.

weight_grid_struct
This is a structure with the following fields
Type Field Description
int weight_id    The weight_id for this weighting.
int age_id    The age_id for this weight value
int time_id    The time_id for this weight value
double weight    The weight value

weight_grid
The return value weight_grid has prototype
     CppAD::vector<weight_grid_struct>  weight_grid
 For each weight_grid_id ,
     weight_grid[weight_grid_id]
 is the information for a specific weighting and specific age and time.

Example
The file get_weight_grid_xam.cpp contains an example that uses this function.
