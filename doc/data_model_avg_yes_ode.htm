One Average Integrand That Requires the ODE

Syntax
avg = data_object.avg_yes_ode(subset_id, pack_vec)

data_object
This object has prototype
     const data_model data_object
 see data_object constructor .

Float
The type Float must be one of the following: double, AD<double>, or a2_double .

subset_id
This argument has prototype
     size_t subset_id
 and is the subset_id we are computing the average integrand for.

Integrand
The integrand_id corresponding to this subset_id must be one of the following: susceptible_enum, withC_enum, prevalence_enum, Tincidence_enum, mtspecific_enum, mtall_enum, mtstandard_enum.

pack_vec
This argument has prototype
     const CppAD::vector<Float>& pack_vec
 and is all the model_variables in the order specified by pack_info . Only the following subvectors of pack_vec are used: pack_info_rate_info , pack_info_mulcov_rate .

avg
The return value has prototype
     Float avg
 This is the average integrand for the specified data point.

Example
The file avg_yes_ode_xam.cpp contains an example and test of using this routine.
