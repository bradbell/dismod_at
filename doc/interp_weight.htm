Interpolate Weight Grid

Syntax
weight = interp_weight(
     age, time, w_info, age_table, time_table, i_wi, j_wi
)

age
This argument has prototype
     double age
 and is the age at which we are interpolating the weight grid values.

time
This argument has prototype
     double time
 and is the time at which we are interpolating the weight grid values.

w_info
This argument has prototype
     const weight_info& w_info
 and is the weight grid. We use the following notation below:
     n_age       = w_info.age_size()
     n_time      = w_info.time_size()
age_table
This argument has prototype
     const CppAD::vector<double>& age_table
 and is the age values corresponding to the weight grid. We use the following notation below: For i_wi = 0 , ..., n_age-1 ,
     a(i_wi) = age_table [ w_info.age_id(i_wi) ]
time_table
This argument has prototype
     const CppAD::vector<double>& time_table
 and is the time values corresponding to the weight grid. We use the following notation below: For j_wi = 0 , ..., n_time-1 ,
     t(j_wi) = time_table [ w_info.time_id(j_wi) ]
i_wi
This argument has prototype
     size_t& i_wi
 and is between 0 and n_age-1 inclusive. Its input value is a hint as to where to start searching for
     a(i_wi) <= age < a(i_wi+1)
 Its output value satisfies the condition above, or is as close as possible under the limits for i_wi .

j_wi
This argument has prototype
     size_t& j_wi
 and is between 0 and n_time-1 inclusive. Its input value is a hint as to where to start searching for
     t(j_wi) <= time < t(j_wi+1)
 Its output value satisfies the condition above, or is as close as possible under the limits for j_wi .

weight
The return value has prototype
     double weight
 It is the value of the bilinear interpolant at the specified age and time .
