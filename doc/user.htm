Note the each of the examples below can by run with the command
     python3_executable example/user/name
 where python3_executable is the python3 program and name is the name of the example (including the .py extension at the end but not the user_ at the front). Running the example will create the database
     build/example/user/example.db
 which can be inspected using any sqlite viewer; e.g., firefox .
