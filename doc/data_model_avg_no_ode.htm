One Average Integrand That Doesn't Require the ODE

Syntax
avg = data_object.avg_no_ode(subset_id, pack_vec)

data_object
This object has prototype
     const data_model data_object
 see data_object constructor .

Float
The type Float must be one of the following: double or one of the ad_types .

subset_id
This argument has prototype
     size_t subset_id
 and is the subset_id we are computing the average integrand for.

Integrand and Rates
The integrand_id corresponding to this subset_id must be one of those listed in the table below. In addition, depending on the integrand, only the corresponding pack_info_rate_info and pack_info_mulcov_rate subvectors of pack_vec are used:
Integrand Rates
Sincidence_enum iota_enum
remission_enum rho_enum
mtexcess_enum chi_enum
mtother_enum omega_enum
mtwith_enum chi_enum, omega_enum
relrisk_enum chi_enum, omega_enum

pack_vec
This argument has prototype
     const CppAD::vector<Float>& pack_vec
 and is all the model_variables in the order specified by pack_info .

avg
The return value has prototype
     Float avg
 This is the average integrand for the specified data point.

Example
The file avg_no_ode_xam.cpp contains an example and test of using this routine.
