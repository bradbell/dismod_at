The Age Table

age_id
This column has type integer and is the primary key for the this table. Its initial value is zero, and it increments by one for each row.

age
This column has type real and is a set of age values. The values in the age column must be unique; i.e., none of the values may repeated.

Ode Grid
Note that the maximum minus the minimum age divided by the ode_step_size will determine the number of age points in the grid used to integrate the ordinary differential equation .

Example
The files age_table.py and get_age_table_xam.cpp contain example age tables.
