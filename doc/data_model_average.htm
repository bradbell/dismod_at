Data Model: Compute One Average Integrand

Syntax
data_object
Float
subset_id
pack_vec
avg
Example

Syntax
avg = data_object.average(subset_id, pack_vec)

data_object
This object has prototype
     data_model data_object
 see data_object constructor . The object data_object is effectively const.

Float
The type Float must be double or a1_double .

subset_id
This argument has prototype
     size_t subset_id
 and is the subset_id we are computing the average integrand for.

pack_vec
This argument has prototype
     const CppAD::vector<Float>& pack_vec
 and is all the model_variables in the order specified by pack_info .

avg
The return value has prototype
     Float avg
 This is the average integrand for the specified data point.

Example
The files avg_no_ode_xam.cpp and avg_yes_ode_xam.cpp contain examples using this routine.
