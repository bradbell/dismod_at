C++: Execution of an SQL Command

Syntax
exec_sql_cmd(db, sql_cmd)

Purpose
Executes an SQL command that does not return any information (and check for an error).

db
The argument db has prototype
     sqlite3* db
sql_cmd
This argument has prototype
     std::string& sql_cmd
 It contains the command that is executed.

Error
If an error occurs, and error message is printing and this routine aborts; i.e., it does not return if an error occurs.
