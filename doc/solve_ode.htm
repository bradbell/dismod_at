Solving the Dismod Ordinary Differential Equation
.

Syntax
solve_ode(
     i_max     ,
     j_max     ,
     step_size ,
     pini      ,
     iota      ,
     rho       ,
     chi       ,
     omega     ,
     S_out     ,
     C_out
)

2DO
Remove i_max , j_max from the arguments to solve_ode because age and time do not explicitly appear in the ODE.

Purpose
This routine determines the value of @(@ C(a,t) @)@ and @(@ S(a, t) @)@ that solve the dismod differential equation for a subset of the ode_grid corresponding to a cohort (constant time minus age).

Notation
We use a_min for the minimum age in age_table and t_min for the minimum time in time_table .

Float
The type Float must be double or a1_double .

i_max
This argument has prototype
     size_t i_max
 It is the maximum ode age grid index for this cohort.

j_max
This argument has prototype
     size_t j_max
 It is the maximum ode time grid index for this cohort.

step_size
This argument has prototype
     const Float& step_size
 and is the value of ode_step_size in the fit command.

pini
This argument has prototype
     const Float& pini
 and is the initial prevalence for this cohort; see pini .

rate
For rate equal to iota , rho , chi and omega , this argument has prototype
     const CppAD::vector<Float>& rate
 and size i_max+1 . For k = 0 , ..., i_max ,
     rate[ k ]
 is the value of the corresponding rate at age
     a = a_min + k * step_size
 and at time
     t = t_min + (j_max - i_max + k) * step_size
S_out, C_out
These arguments have prototypes
     CppAD::vector<Float>& S_out
     CppAD::vector<Float>& C_out
 and their input sizes are zero. Upon return they have size is i_max + 1 and for k = 0 , ..., i_max ,
     S_out[ k ]  ,  C_out[ k ]
 is the value of @(@ S(a,t) @)@ and @(@ C(a,t) @)@
     a = a_min + k * step_size
 and at
     t = t_min + (j_max - i_max + k) * step_size
Example
The file solve_ode_xam.cpp contains and example and test of solve_ode. It returns true for success and false for failure.
