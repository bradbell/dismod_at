Purpose
The command
     dismod_at database sample asymptotic
 samples form an asymptotic approximation for the posterior distribution of the model_variables .

Notation
iota_n model incidence for north_america
u_m incidence random effect for mexico
u_c incidence random effect for canada
y_n measured incidence for north_america
y_m measured incidence for mexico
y_c measured incidence for canada
s_n standard deviation for y_n
s_m standard deviation for y_m
s_c standard deviation for y_c
s_r standard deviation for random effects

Likelihood
We define @(@ h(y, \mu , \sigma) @)@ to be the log-density for a @(@ \B{N}( \mu, \sigma^2 ) @)@ distribution; i.e., @[@ h(y, \mu, \sigma) = - \frac{ ( y - \mu )^2 }{ \sigma^2 } - \log \left( \sigma \sqrt{ 2 \pi } \right) @]@ The total log-likelihood for this example is @[@ h[ y_n, \iota_n, s_n ] + h[ y_m, \exp( u_m ) \iota_m, s_m ] + h[ y_c, \exp( u_c ) \iota_c, s_c ] + h( u_m, 0, s_r ) + h( u_c , 0 , s_r ) @]@

# no need to include sqrt{2 \pi} term (it does not depend on model variables)
def h(y, mu, sigma ) :
     if sigma <= 0.0 :
          return - float("inf")
     res = (y - mu ) / sigma
     return - res * res - math.log( sigma )

     ret   += h(y_m, math.exp(u_m) * iota_n, s_m )
     ret   += h(y_c, math.exp(u_c) * iota_n, s_c )
     ret   += h(u_m, 0.0, s_r ) + h(u_c, 0.0, s_r)
     return ret

     value     = var_avg[ node_name2var_id[node_name] ]
     check     = x_avg_mcmc[i]

     if abs( avg_diff ) > 0.05 :
          print('avg_diff = ', avg_diff, ', random_seed = ', random_seed )

     value     = var_std[ node_name2var_id[node_name] ]

     std_diff  = check / value - 1.0
     # This is a small sample case (only three data points)
     # so we do not expect the asymptotic statistics to be correct.
     # Note that in this case, the asymptotics are an over estimate.
