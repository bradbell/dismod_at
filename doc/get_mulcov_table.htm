C++: Get the Covariate Multiplier Table Information

Syntax
Purpose
db
mulcov_struct
     mulcov_type_enum
mulcov_table
Example

Syntax
mulcov_table = get_mulcov_table(db)

Purpose
To read the mulcov_table and return it as a C++ data structure.

db
The argument db has prototype
     sqlite3* db
 and is an open connection to the database.

mulcov_struct
This is a structure with the following fields
Type Field Description
mulcov_type_enum    mulcov_type    See the description of mulcov_type below.
int rate_id    The rate_id for this multiplier
int integrand_id    The integrand_id for this multiplier
int covariate_id    The covariate_id for this multiplier
int smooth_id    The smooth_id for this multiplier

mulcov_type_enum
The mulcov_type_enum possible values are rate_value_enum, meas_value_enum, and meas_noise_enum. These correspond to the values in the mulcov_type column of the mulcov_type table.

mulcov_table
The return value mulcov_table has prototype
     CppAD::vector<mulcov_struct>  mulcov_table
 For each mulcov_id ,
     mulcov_table[mulcov_id]
 is the information for the corresponding multiplier.

Example
The file get_mulcov_table_xam.cpp contains an example that uses this function.
