C++: Get the Smoothing Grid Information

Syntax
Purpose
db
smooth_grid
smooth_grid_struct
Check
Example

Syntax
smooth_grid = get_smooth_grid(db)

Purpose
To read the smooth_grid_table and return it as a C++ data structure.

db
The argument db has prototype
     sqlite3* db
 and is an open connection to the database.

smooth_grid
The return value smooth_grid has prototype
     CppAD::vector<smooth_grid_struct>  smooth_grid
 For each smooth_grid_id ,
     smooth_grid[smooth_grid_id]
 is the information for a specific smoothing and specific age and time.

smooth_grid_struct
This is a structure with the following fields
Type Field Description
int smooth_id    The smooth_id for this smoothing.
int age_id    The age_id for this grid point
int time_id    The time_id for this grid point
int value_prior_id    The value_prior_id for this smoothing, age, and time.
int dage_prior_id    The dage_prior_id for this smoothing, age, and time.
int dtime_prior_id    The dtime_prior_id for this smoothing, age, and time.
int value_prior_id    The const_value for this smoothing, age, and time.

Check
The values in the smooth_grid_table are checked to make sure that const_value is null or value_prior_id is null and not both are null.

Example
The file get_smooth_grid_xam.cpp contains an example that uses this function.
