C++: Get the Time Table Information

Syntax
Purpose
db
time_table
Example

Syntax
time_table = get_time_table(db)

Purpose
To read the time_table and return it as a C++ data structure.

db
The argument db has prototype
     sqlite3* db
 and is an open connection to the database.

time_table
The return value time_table has prototype
     CppAD::vector<double>  time_table
 For each time_id ,
     time_table[time_id]
 is the time for the corresponding time .

Example
The file get_time_table_xam.cpp contains an example that uses this function.
