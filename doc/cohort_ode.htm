void cohort_ode(

     const CppAD::vector<Float>&  rho       ,

     const CppAD::vector<Float>&  omega     ,

     CppAD::vector<Float>&        c_out     )

Purpose
This routine approximates the functions @(@ S(a) @)@ and @(@ C(a) @)@ that solve the ordinary differential equation . The functions @(@ S(a, t) @)@, @(@ C(a, t) @)@ satisfy this equation along cohort lines were @(@ t - a @)@ is constant.

Float
The type Float must be double or a1_double .

n_cohort
We use the notation n_cohort for the number of grid points along this cohort at which the approximate solution is returned.

rate_case
This is the value of rate_case in the option table.

age
This vector has size n_cohort and specifies the age values for this cohort. For k = 1 , ..., n_cohort-1 , the rates are approximated as constant over the k-th age interval from age[k-1] to age[k] .

pini
This is the prevalence corresponding to age[0] ; see pini . At this age @(@ a_0 @)@, @(@ C( a_0 ) @)@ is equal to pini and @(@ S( a_0 ) = 1 - C( a_0 ) @)@.

rate
For rate equal to iota, rho, chi, and omega, we use rate for the corresponding argument. It is vector with size n_cohort and rate5[k] specifies the corresponding rate at age age[k] . s_out This vector has size n_cohort . The input value of its elements does not matter. Upon return, s_out[k] is the approximation solution for @(@ S(a, t) @)@ at the corresponding age and time. c_out This vector has size n_cohort . The input value of its elements does not matter. Upon return, c_out[k] is the approximation solution for @(@ C(a, t) @)@ at the corresponding age and time.

Example
The file cohort_ode_xam.cpp contains and example and test of cohort_ode. It returns true for success and false for failure.
