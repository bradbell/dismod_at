The Time Table

time_id
This column has type integer and is the primary key for the this table. Its initial value is zero, and it increments by one for each row.

time
This column has type real and is a set of time values. The values in the time column must be unique; i.e., none of the values may be repeated.

Ode Grid
Note that the maximum minus the minimum time divided by the ode_step_size will determine the number of time points in the grid used to integrate the ordinary differential equation .

Example
The files time_table.py and get_time_table_xam.cpp contain example time tables.
