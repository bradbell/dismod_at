Run optimization to determine the optimal fixed and random effects

Syntax
fit_object.run_fit(random_only, option_map)
fit_object
This object must have been constructed with fit_or_sample equal to fit.

Scaling
During optimization the fixed effects , that have eta not null in their value prior, are scaled; see scaling .

random_only
If this argument is true, the fixed effects are set to their starting value and only the random effects are optimized. Otherwise, both the fixed and random effects are optimized.

option_map
This argument is effectively const and must have the following values: For name equal to derivative_test , tolerance , max_num_iter , print_level , accept_after_max_steps , and for fit equal to fixed and random
     option_map["name_fit"]
 is the value in the option_table for the corresponding option. It must also have
     option_map["bound_frac_fixed"]
     option_map["limited_memory_max_history_fixed"]
