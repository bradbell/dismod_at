C++: Get the Prior Table Information

Syntax
Purpose

Example

Syntax
prior_table = get_prior_table(db)

Purpose
To read the prior_table and return it as a C++ data structure.

db
The argument db has prototype
     sqlite3* db
 and is an open connection to the database.

prior_table
The return value prior_table has prototype
     CppAD::vector<prior_struct> prior_table
 For each prior_id ,
     prior_table[prior_id]
 is the information for the corresponding prior.

prior_struct
This is a structure with the following fields
Type Field Description
std::string prior_name The prior_name for this prior
int density_id The density_id for this prior
double lower The lower limit for this prior
double upper The upper limit for this prior
double mean The mean for this prior
double std The std for this prior
double eta The eta for this prior double nu The nu for this prior

Checks
The values in the prior_table are checked to make sure that:
  1. The limits and mean satisfy lower <= mean <= upper .
  2. The standard deviation std > 0 (except for the uniform density case).
  3. In the log density cases eta is not null and eta > 0 .
  4. In the Student density cases nu is not null and nu > 2 .


Example
The file get_prior_table_xam.cpp contains an example that uses this function.
