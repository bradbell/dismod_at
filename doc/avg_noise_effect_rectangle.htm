Prototype 

template <class Float>
Float avg_noise_effect::rectangle(
     double                           age_lower        ,
     double                           age_upper        ,
     double                           time_lower       ,
     double                           time_upper       ,
     size_t                           weight_id        ,
     size_t                           integrand_id     ,
     const CppAD::vector<double>&     x                ,
     const CppAD::vector<Float>&      pack_vec         ,

age_lower
the lower age in the rectangle.

age_upper
the upper age in the rectangle; age_lower <= age_upper .

time_lower
the lower time in the rectangle.

time_upper
the upper time in the rectangle; time_lower <= time_upper .

weight_id
This is the weight_id in the weight table corresponding to this average integrand.

integrand_id
This is the integrand_id in the integrand table corresponding to this average integrand.

x
This is the vector of covariates for this average.

Float
The type Float must be double or a1_double .

pack_vec
is all the model_variables in the order specified by pack_object .

avg
The return value avg is the average of the integrand using the specified weighting over the specified rectangle

Example
The file avg_noise_effect_xam.cpp contains an example and test of using this routine.
