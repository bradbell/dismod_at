The Density Table
The density table is used to identify densities. It has the following columns:

density_id
This column has type integer and is the primary key for this table. Its initial value is zero, and it increments by one for each row.

density_name
The names listed below and only the those names, must appear in the density_name column. In addition, they must appear in the order below.. Note that density_name is unique; i.e., each name only appears once.

uniform
Uniform

gaussian
Gaussian

laplace
Laplace

students
Student's-t

log_gaussian
Log-Gaussian

log_laplace
Log-Laplace

log_students
Log-Student's-t

Example
The files density_table.py and get_density_table_xam.cpp contain an example density tables.
