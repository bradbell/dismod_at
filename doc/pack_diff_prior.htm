Difference Priors For Packed Variables

Syntax
diff_prior = pack_diff_prior(pack_object, s_info_vec)

pack_object
This argument has prototype
     const pack_info& pack_object
 and is the pack_info information corresponding to the model_variables .

s_info_vec
This argument has prototype
     const CppAD::vector<smooth_info>& s_info_vec
 For each smooth_id ,
     s_info_vec[ smooth_id ]
 is the corresponding smooth_info information.

diff_prior
The return value has prototype
     CppAD::vector<diff_prior_struct> diff_prior
 It contains one diff_prior_struct for each pair of packed variable indices that have a dage or dtime prior. Let n_diff_prior = diff_prior.size() . For k = 0 , ..., n_diff_prior-1 :
diff_prior[k].plus_var_id packed variable index of positive term
diff_prior[k].minus_var_id packed variable index of negative term
diff_prior[k].prior_id primary key in prior table for this difference

Example
The file pack_diff_prior_xam.cpp contains an example and test that uses this routine.
