Purpose
Extracts the information for one weighting from the weight_grid_table . In addition, this routine checks the weight_info table rectangular grid assumption.

w_default
This is the default constructor. It can be used to create an empty weight_info object that is later set equal to another weight_info object. This is useful when creating vectors of such objects.

w_info
In all its uses, except during construction, this object has prototype
     const weight_info w_info
 The mean of the corresponding constructor arguments are specified below:

w_test
This object is the result of the testing constructor and can be used the same as w_info .

age_table
This argument has prototype
     const CppAD::vector<double>& age_table
 and is the age_table .

time_table
This argument has prototype
     const CppAD::vector<double>& time_table
 and is the time_table .

weight_table
This argument has prototype
     const CppAD::vector<double>& weight_table
 and is the weight_table .

weight_grid_table
This argument has prototype
     const CppAD::vector<weight_grid_struct>& weight_grid_table
 an is the weight_grid_table .

age_id
This argument has prototype
     const CppAD::vector<size_t>& age_id
 It specifies the age grid indices; i.e.
     w_info.age_id(i) = age_id[i]
time_id
This argument has prototype
     const CppAD::vector<size_t>& time_id
 It specifies the time grid indices; i.e.
     w_info.time_id(i) = time_id[i]
weight
This argument has prototype
     const CppAD::vector<double>& weight
 It specifies the weight grid values in row major order; i.e.
     w_info.weight(i, j) = time_id[i*n_time + j]
n_age
This result has prototype
     size_t n_age
 and is the number of age values for this weighting.

n_time
This result has prototype
     size_t n_time
 and is the number of time values for this weighting.

i
The argument i has prototype
     size_t i
 and is the age index; i < n_age .

j
The argument j has prototype
     size_t j
 and is the time index; j < n_time .

a_id
This return value has prototype
     size_t a_id
 and is the i-th age_id for this weighting and increases with i ; i.e., for i < n_age-2
     w_info.age_id(i) < w_info.age_id(i+1)
t_id
This return value has prototype
     size_t t_id
 and is the j-th time_id for this weighting and increases with j ; i.e., for j < n_time-2
     w_info.time_id(j) < w_info.time_id(j+1)
w
This return value has prototype
     double w
 and is the weighting for the corresponding age and time indices.

Example
The file weight_info_xam.cpp contains an example that uses this function.
