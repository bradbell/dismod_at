Interpolation from Smoothing to Ode Grid

Syntax
smooth2ode si2ode(
     n_age_ode, n_time_ode, ode_step_size , age_table, time_table,
     s_info
)
var_ode = si2ode.interpolate( var_si, ode_index )

Float
The type Float must be one of the following: double or one of the ad_types .

smooth2ode
This constructs an object that interpolates from the specified smoothing grid to the ode grid.

n_age_ode
This argument has prototype
     size_t n_age_ode
 It is the number of points in the ode age grid .

n_time_ode
This argument has prototype
     size_t n_time_ode
 It is the number of points in the ode time grid .

ode_step_size
This argument has prototype
     double ode_step_size
 and is the value of ode_step_size in the fit command.

age_table
This argument has prototype
     const CppAD::vector<double>& age_table
 and is the age values corresponding to the age_id values.

time_table
This argument has prototype
     const CppAD::vector<double>& time_table
 and is the time values corresponding to the time_id values.

s_info
This argument has prototype
     const smooth_info& s_info
 and is the smoothing grid. We use the following notation below:
     n_age_si       = s_info.age_size()
     n_time_si      = s_info.time_size()
 The only other s_info functions that are used are used by smooth2ode are: s_info.age_id and s_info.time_id ,

interpolate
This is a const function that interpolates from the smoothing grid to the ode grid.

var_si
This argument has prototype
     const CppAD::vector<Float>& var_si
 and its size is n_age_si*n_time_si . For i_si = 0 , ..., n_age_si-1 , j_si = 0 , ..., n_time_si-1 ,
     var_si[ i_si * n_time_si + j_si ]
 is the value of the variable (being interpolated) at
     age  = age_table[  s_info.age_id(i_si) ]
     time = time_table[ s_info.time_id(j_si) ]
ode_index
This argument has prototype
     const CppAD::vector<size_t>& ode_index
 For k = 0 , ..., ode_index.size() the corresponding ode grid point age a_i is defined by
     i(k) = int( ode_index[k] / n_time_ode )
 and the ode grid point time t_j is defined by
     j(k) = mod( ode_index[k], n_time_ode )
var_ode
This return value has prototype
     const CppAD::vector<Float>& var_ode
 and its size is ode_index.size() . For k = 0 , ..., ode_index.size()-1 ,
     var_ode[ k ]
 is the value of the interpolated value for the variable at
     age  = a_i =  age_min + i(k) * ode_step_size
     time = t_j =  time_min + j(k) * ode_step_size
 where age_min ( time_min ) is the minimum value in age_table ( time_table ).
