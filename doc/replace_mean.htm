Replace Prior Means

Syntax
prior_object.replace_mean(prior_mean) .

Prototype 


void prior_model::replace_mean(const CppAD::vector<double>& prior_mean)

Purpose
By default, the mean values in the prior table are used when computing the prior for the fixed and random . This routine can be used to replace these means (which is useful for posterior simulation ). This object has prototype
     const prior_model prior_object
prior_mean
This vector has size n_var*3 where n_var is the number of variables in the var_table . For var_id equal zero to n_var-1 ,
     prior_mean[ 3 * var_id + k ]
 replace the mean in the value prior ( k=0 ), dage prior ( k=1 ), dtime prior ( k=2 ). This value is not used when the corresponding prior_id is null.
