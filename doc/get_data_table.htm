Syntax
get_data_table(
     db, n_covariate, age_min, age_max, time_min, time_max,
     data_table, data_cov_value
)

Purpose
To read the data_table and return it as a C++ data structure.

db
The argument db has prototype
     sqlite3* db
 and is an open connection to the database.

n_covariate
This argument has prototype
     size_t n_covariate
 and is the size of the covariate_table .

age_min
This argument has prototype
     double age_min
 and is the minimum value in the age_table . If an age_lower value is less than age_min , an error is reported.

age_max
This argument has prototype
     double age_max
 and is the maximum value in the age_table . If an age_upper value is greater than age_max , an error is reported. The condition
     age_lower <= age_upper
 is also checked.

time_min
This argument has prototype
     double time_min
 and is the minimum value in the time_table . If an time_lower value is less than time_min , an error is reported.

time_max
This argument has prototype
     double time_max
 and is the maximum value in the time_table . If an time_upper value is greater than time_max , an error is reported. The condition
     time_lower <= time_upper
 is also checked.

data_table
This argument has prototype
     CppAD::vector<data_struct>&  data_table
 On input its size is zero and upon return it has one element for each row in the data table. For each data_id ,
     data_table[data_id]
 is the data_struct information for the corresponding data.

data_struct
This is a structure with the following fields
Type Field Description int integrand_id The integrand_id for this measurement
int density_id The density_id for this measurement
int node_id The node_id for this measurement
int weight_id The weight_id for this measurement
int hold_out The hold_out for this measurement
double meas_value The meas_value for this measurement
double meas_std The meas_std for this measurement
double eta The eta for this measurement
double nu The nu for this measurement
double age_lower The age_lower for this measurement
double age_upper The age_upper for this measurement
double time_lower The time_lower for this measurement
double time_upper The time_upper for this measurement

data_cov_value
This argument has prototype
     CppAD::vector<double>&  data_cov_value
 On input its size is zero. Upon return, its size is the number of data points times the number of covariates. For each covariate_id and data_id pair
     data_cov_value[data_id * n_covariate + covariate_id]
 is the corresponding covariate value.

Example
The file get_data_table_xam.cpp contains an example that uses this function.
