C++: Get the Rate Table Information

Syntax
rate_table = get_rate_table(db)
 rate_name = get_rate_name(rate_id)

get_rate_table
Reads the rate_table and return it as a C++ data structure.

get_rate_name
Sets rate_name to a std::string that corresponds to rate_id .

db
The argument db has prototype
     sqlite3* db
 and is an open connection to the database.

rate_struct
This is a structure with the following fields
Type Field Description
rate_enum rate enum corresponding to rate_name
int parent_smooth_id The parent_smooth_id
int child_smooth_id The child_smooth_id
int child_nslist_id The child_nslist_id

rate_table
The return value rate_table has prototype
     CppAD::vector<rate_struct>  rate_table
 For each rate_id ,
     rate_table[rate_id]
 is the information value for the corresponding rate_id .

rate_enum
The rate_table can also be addressed using the rate_enum values as follows:
rate_id    rate_name
pini_enum    pini
iota_enum    iota
rho_enum    rho
chi_enum    chi
omega_enum    omega
The number of these enum values is number_rate_enum.
