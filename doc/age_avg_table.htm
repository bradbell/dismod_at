Creation
This table changes each time the ode_step_size or the age_avg_split changes. These value can change without having to re-execute the init_command . For this reason, this table is re-written for each command with the exception of the following commands: set , db2csv , csv2db , and modify .

age_avg_id
This column has type integer and is the primary key for the this table. Its initial value is zero, and it increments by one for each row.

age
This column has type real and is a monotone increasing values in the age average grid The values are unique; i.e., none of the values are repeated.
