Open An Sqlite3 Database Connection

Syntax
db = open_connection(file_name, new_file)
file_name
This argument has prototype
     const std::string& file_name
 It is the name of the file where the data base is (or will be) stored.

new_file
This argument has prototype
     bool new_file
 If it is true, and a database with the same name already exists, the existing database is deleted before creating the connection.

db
The return value has prototype
     sqlite3* db
 and is the database connection.

close
When you are done with db , you must call
     sqlite3_close(db)
