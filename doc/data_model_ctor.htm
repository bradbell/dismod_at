data_object
This is the data_model object being constructed.

meas_noise_effect
This is the value of meas_noise_effect in the option table.

rate_case
This is the value of rate_case in the option table.

bound_random
This is the bound_random .

n_covariate
This is the number of covariates; i.e., number or rows in covariate_table .

ode_step_size
This is the ode_step_size .

age_avg_grid
This is the age average grid .

age_table
This is the age_table .

time_table
This is the time_table .

integrand_table
This is the integrand_table .

subset_object
This is the sub-sampled version of the data or avgint table; see data_subset_obj , avgint_subset_obj .

subset_cov_value
This is the sub-sampled version of the covariates; see data_subset_cov_value , avgint_subset_cov_value . A reference to subset_cov_value is used by data_object (so age_table cannot be deleted for as long as data_object is used).

w_info_vec
For each weight_id ,
     w_info_vec[ weight_id ]
 is the corresponding weight_info information.

s_info_vec
For each smooth_id ,
     s_info_vec[ smooth_id ]
 is the corresponding smooth_info information. For each s_info_vec[smooth_id] object, only the following functions are used: age_size, time_size, age_id, time_id.

pack_object
This is the pack_info information corresponding to the model_variables . A reference to pack_object is used by data_object .

child_object
This is the child_info information corresponding to parent node, node table, and data table.

n_covariate_
set to n_covariate .

ode_step_size_
set to ode_step_size .

pack_object_
set to pack_object .

minimum_meas_cv_
Set to a vector with the same size as integrand_table and
     minimum_meas_cv_[integrand_id] =
          integrand_table[integrand_id].minimum_meas_cv
replace_like_called_
initialize as false.

n_child_
set to number of children.

data_subset_obj_
for each subset_id , set data_subset_obj_[subset_id] fields that are command both data_subset and avgint_subset.

child_ran_zero_
for each child < n_child_ , child_ran_zero_[child] is true if all the random effects for this child are constrained to be zero.

data_info_
for each subset_id , set data_info_[subset_id] is extra information for each data point.

avgint_obj_
The avg_integrand constructor is used to create this argument. The arguments to this constructor are a subset of the arguments to the data_model constructor and are passed through (with the same name) to the avg_integrand constructor.
