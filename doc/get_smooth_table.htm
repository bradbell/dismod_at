C++: Get the Smoothing Table Information

Syntax
smooth_table = get_smooth_table(db)

Purpose
To read the smooth_table and return it as a C++ data structure.

db
The argument db has prototype
     sqlite3* db
 and is an open connection to the database.

smooth_table
The return value smooth_table has prototype
     CppAD::vector<smooth_struct>  smooth_table
 For each smooth_id ,
     smooth_table[smooth_id]
 is the information for the corresponding smooth_id .

smooth_struct
This is a structure with the following fields
Type Field Description
std::string    smooth_name The smooth_name for this smoothing.
int n_age The n_age for this smoothing.
int n_time The n_time for this smoothing.
int mulstd_value_prior_id The mulstd_value_prior_id for this smoothing.
int mulstd_dage_prior_id The mulstd_dage_prior_id for this smoothing.
int mulstd_dtime_prior_id The mulstd_dtime_prior_id for this smoothing.
