Plot The Data Fit By Integrand

Syntax
database
integrand_list
pdf_file
n_point_list
Example

Syntax
n_point_list = plot_data_fit(database, integrand_list, pdf_file)
database
This str is the file name for an existing dismod_at database that contains the results of a fit_command .

integrand_list
Each element of this list is a str containing an integrand_name that we are plotting the fit for.

pdf_file
Is the location where the pdf file containing the plot will be placed.

n_point_list
The i-th element of this list is the number of rows in the data_subset_table that correspond to the i-th integrand in integrand_list . If this is less than 2, the i-th integrand is not included in the plots.
