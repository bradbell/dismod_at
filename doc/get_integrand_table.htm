C++: Get the Integrand Table Information

Syntax
integrand_table = get_integrand_table(db)

Purpose
To read the integrand_table and return it as a C++ data structure.

db
The argument db has prototype
     sqlite3* db
 and is an open connection to the database.

integrand_table
The return value integrand_table has prototype
     CppAD::vector<integrand_enum>  integrand_table
 For each integrand_id ,
     integrand_table[integrand_id]
 is the corresponding integrand described below:

integrand_enum
This is an enum type with the following values:
integrand_enum    integrand_name
Sincidence_enum    Sincidence
Tincidence_enum    Tincidence
remission_enum    remission
mtexcess_enum    mtexcess
mtother_enum    mtother
mtwith_enum    mtwith
susceptible_enum    susceptible
withC_enum    withC
prevalence_enum    prevalence
mtspecific_enum    mtspecific
mtall_enum    mtall
mtstandard_enum    mtstandard
relrisk_enum    relrisk

integrand_enum2name
This is a global variable. If integrand , is an integrand_enum value, integrand_enum2name[integrand] is the integrand_name corresponding to the enum value.

Example
The file get_integrand_table_xam.cpp contains an example that uses this function.
