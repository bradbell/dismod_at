The Random Effects Value Density Function

c
We use @(@ C @)@ to denote the size of the child group corresponding to the parent_node_id . For @(@ c = 0 , \ldots , C-1 @)@, we use @(@ c @)@ to denote the corresponding child.

j
For each rate_id there is a corresponding child_smooth_id . For each child_smooth_id there is a corresponding number of age points n_age and number of time points n_time . We use the single index @(@ j @)@ (instead of three indices) to denote a each possible combination of rate, age, and time index.

k_j
We use @(@ k_j @)@ to denote the rate_id corresponding to index @(@ j @)@.

a_j
We use @(@ a_j @)@ to denote the age corresponding to index @(@ j @)@.

t_j
We use @(@ t_j @)@ to denote the time corresponding to index @(@ j @)@.

u_c,j
We use @(@ u @)@ for the vector of the random effects and @(@ u_{c,j} @)@ the component for the child specified by @(@ c @)@ and the rate, age and time specified by @(@ j @)@.

lambda_j
We use @(@ \lambda_j @)@ for the mulstd_value_prior_id multiplier for the child_smooth_id corresponding to the rate specified by @(@ j @)@.

delta_j
We use @(@ \delta_j @)@ to denote the standard deviation std corresponding to the child_smooth_id , the value_prior_id , and the rate, age, and time specified by j .

d_j
We use @(@ d_j @)@ to denote the density_id and eta corresponding to j .

mu_j
We use @(@ \mu_j @)@ for the mean corresponding to the value_prior_id which must be zero; see the child smoothing restrictions.

V_c,j^u
The definition of the value density function for @(@ u_{c,j} @)@ is @[@ V_{c,j}^u ( u | \theta ) = \exp \left[ D ( u_{c,j} \W{,} \mu_j \W{,} \lambda_j \delta_j \W{,} d_j ) \right] @]@ where @(@ D @)@ is the log-density function . As noted above, @(@ \mu_j @)@ must be zero, @(@ d_j:name @)@ must be gaussian, and @(@ \eta_j @)@ does not matter.

V^u
Let @(@ m @)@ be the number of random effects. The value density for all the random effects @(@ u @)@ is defined by @[@ V^u ( u | \theta ) = \prod_{c=0}^{C-1} \prod_{j=0}^{m-1} V_{c,j}^u ( u | \theta ) @]@
