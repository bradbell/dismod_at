Split String Into Substrings With Spaces as Delimiter

Syntax
vec = split_space(str)

str
This is the string that we are splitting. Each substring is non-empty with by one or more spaces between them.

vec
This is the resulting vector of non-empty substrings. If all the characters in str are spaces, the resulting vector with have size zero.

Example
The file split_space_xam.cpp contains an example and test of using this routine.

Prototype 

CppAD::vector<std::string> split_space(const std::string& str)
