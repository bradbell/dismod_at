Bilinear Interpolation

Syntax
z = bilinear_interp(
     x, y, x_grid, y_grid, z_grid, i, j
)

x
This argument has prototype
     double x
 and is the x value at which we are interpolating the bilinear function.

y
This argument has prototype
     double y
 and is the y value at which we are interpolating the bilinear function.

x_grid
This argument has prototype
     const CppAD::vector<double>& x_grid
 and is the x values corresponding to the z_grid . We use the following notation:
     n_x = x_grid.size()
 Note that n_x must be greater than or equal one.

y_grid
This argument has prototype
     const CppAD::vector<double>& y_grid
 and is the y values corresponding to the z_grid . We use the following notation:
     n_y = y_grid.size()
 Note that n_y must be greater than or equal one.

z_grid
This argument has prototype
     const CppAD::vector<double>& z_grid
 and has size n_x * n_y . For i = 0 , ..., n_x-1 , j = 0 , ..., n_y-1 ,
     z_grid[ i * n_y + j]
 is the value of the bilinear interpolant @(@ z(x,y) @)@, at x = x_grid[i] and y = y_grid[j] .

i
This argument has prototype
     size_t& i
 and is between 0 and n_x-1 inclusive. Its input value is a hint as to where to start searching for
     x_grid[i] <= x < x_grid[i+1]
 Its output value satisfies the condition above, or is as close as possible under the limits 0 <= i < n_x .

j
This argument has prototype
     size_t& j
 and is between 0 and n_y-1 inclusive. Its input value is a hint as to where to start searching for
     y_grid[j] <= y < y_grid[j+1]
 Its output value satisfies the condition above, or is as close as possible under the limits 0 <= j < n_y .

z
The return value has prototype
     double z
 It is the value of the bilinear interpolant at the specified x and y .
